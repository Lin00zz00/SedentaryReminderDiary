# 每日久坐情况调查

一个简单易用的移动端问卷应用，用于记录每日久坐情况。

## 使用方法

1. 打开网页
2. 选择相应的选项
3. 点击"生成报告并下载"按钮
4. 报告图片会自动保存到手机相册

## 在线访问

访问地址：[https://lin00zz00.github.io/SedentaryReminderDiary/](https://lin00zz00.github.io/SedentaryReminderDiary/)

## 本地运行

直接在浏览器中打开 `index.html` 文件即可使用。
